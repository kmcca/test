Metadata provides detailed information about the application to enable you to analyze the security performance of all the applications in your portfolio. Use the metadata to group and sort your portfolio data by any of the standard categories to measure performance and policy compliance across the portfolio. Use the custom fields to create metadata fields of your own.

<p font-size="13pt"><b>Before You Begin</b></p>

You must have either the Creator or Security Lead [role](https://docs.veracode.com/r/c_role_permissions) to be able to add or edit the metadata of an application.

<p font-size="13pt"><b>Steps</b></p>

1.  In the Veracode Platform, select **My Portfolio** \> **Applications**.

2.  Find the application you want to edit and select **Actions** \> **Edit Application**.

3.  Enter comma-separated tags in the Metadata field or enter values in the custom fields.

    Use the custom fields to collect data that is specific to your data analysis needs.

4.  Click **Submit**.
